# Joyclub-Stylus-AdRemove

A custom [Stylus](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) style for [Joyclub](https://www.joyclub.de) that removes the Joyclub advertisement.

## Installation

1. Install the [Stylus browser extension from the Chrome Web Store](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne).  
2. Download [JSON](joyclub-adremove.json) from this repository.  
3. In Stylus, click the **Manage** button (Stylus icon â†’ "Manage").  
4. Click **Import** in the left-hand menu.  
5. Select the downloaded `joyclub-adremove.json` file.  
6. Enable the imported style.

## Compatibility

This style is designed **specifically for** [https://www.joyclub.de](https://www.joyclub.de) and will not affect other websites.
